🌳 How Recursive Subscribe Works:

  • Discovery: Browses structure recursively to find all variables
  • Smart Batching: Groups subscriptions for better performance
  • Path Tracking: Shows full path for each variable (e.g., Recipe.Settings.Temperature)
  • Depth Control: Prevents infinite recursion with max depth limit
  • Batch Results: Shows success/failure for each variable found
